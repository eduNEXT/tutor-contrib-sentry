sentry plugin for `Tutor <https://docs.tutor.edly.io>`__
###############################################################################

Sentry Integration for Open edX


Installation
************

.. code-block:: bash

    pip install git+https://github.com/eduNEXT/tutor-contrib-sentry

Usage
*****

.. code-block:: bash

    tutor plugins enable sentry
